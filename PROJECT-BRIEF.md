**PROJECT BRIEF – North Hanover Grille Modern Web App**

**Overall Goal**
Replace the very basic, outdated “Windows 95 looking” site with a beautiful, modern, attractive, professional full-stack Progressive Web App (PWA).

**Key Visual & User Experience Requirements**
- Modern, clean, attractive design that flows and attracts customers (dark mode preferred, high-quality photos, smooth animations, mobile-first).
- Professional restaurant feel.

**Public-Facing Features**
- Beautiful menu with Beers on Tap, full menu, daily specials.
- Dynamic beer list with photos, names, prices, types, descriptions.
- PDF menu upload with easy hide/show.
- “Order Now” button (future payment ready).
- All content respects admin hide/show toggles.

**Admin Back Door**
- Simple password-protected login.
- Modern, intuitive interface with clear sections.
- **Photo Management**: Organized folders for Beers, Menu Items, Specials, etc. Easy to add photos from phone and make them live on public site.
- Add/edit/delete beers with photo selection.
- Colorful one-click red/green hide/show toggles.
- PDF upload for daily specials with hide/show.
- All changes appear instantly.

**Technical Requirements**
- 100% pure code.
- Automatic photo upload from phone.
- PWA features.
- Data persistence.

Start building the complete app with admin panel and public menu first. Focus on easy photo management and modern, self-explanatory admin interface.
